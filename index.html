Beehive

Beehive is an e-commerce site that connects consumers with artisians and their unique products. Powered by the Spoonflower API, buyers can customize any item by choosing a unique Spoonflower fabric design. Beehive is a Rails application built in two weeks as a part of The Iron Yard Durham Cohort 5 Demo Day. As the sole front end developer on the team, I was resonsible for all aspects of design and user interface.

Bitter-fit

Bitter-fit is a personalized stat-tracker and collaborative project with the Python class at The Iron Yard. It is a single-page application built using AngularJS over the course of 3 nights. It calls to a Django API built by our Python group members. User's can add daily activities and keep track of their progress. My partner and I were responsible for all things involving the graphic user interface including jQuery, Angular, Sass and HTML.
